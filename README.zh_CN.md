# 玲珑dbus拦截转发

linglong-dbus-proxy 是在 linglong-box 中运行的 linglong 应用程序的 dbus 代理。

## Dependencies

### Build dependencies

* Qt >= 5.10

* libdbus-1-dev

* libgtest-dev

## Installation

### Build from source code

1. Make sure you have installed all dependencies.

2. Build:

    ```bash
    mkdir build
    cd build
    cmake ..
    make
    ```

3. Install:

    ```bash
    sudo make install
    ```
